# 模块 6：AI 模拟面试

**当前状态：可用；实时语音受浏览器、麦克风和系统语音引擎影响。**

## 功能介绍

按简历和目标岗位连续追问，支持标准/挑战/高压追问、实时语音打断、录音转写、评分报告和历史复盘。

## 操作方法

1. 选择是否使用简历；可选已有简历、无简历或上传新简历。
2. 选择已有职位或手工填写目标岗位。
3. 选择面试类型、压力等级和题目数量。
4. 创建面试，允许麦克风权限并开始实时语音；也可使用录音上传回退。
5. 回答、打断、静音、重听或结束。
6. 查看评分、证据、改进建议和历史记录。

## 使用场景

准备自我介绍、岗位专业问题、行为面试、压力追问和复盘表达。

## AI 工作流

岗位/简历快照 → 问题 → 用户语音/文本 → 转写 → 连续追问 → 证据评分 → 幂等报告。Provider 失败时回退本地逻辑。

## 用户输入与输出

- 输入：岗位、简历、面试配置、语音或文本答案。
- 输出：实时字幕、追问、评分维度、证据、总结和历史记录。

## 注意事项

面试录音包含个人信息；仅在可信设备使用。Android TTS 依赖系统语音引擎，部分设备可能无法播放；录音上传曾验证本地 Whisper 中文转写。

## 常见问题

- **实时语音不可用？** 检查麦克风权限、HTTPS/localhost、安全上下文和浏览器支持。
- **AI 没声音？** 使用重听、文本字幕或录音回退。
- **可以随时打断吗？** 支持 barge-in，但取决于浏览器语音识别能力。
