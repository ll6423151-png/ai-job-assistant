# 模块 12：帮助与反馈

**当前状态：本次新增。**

## 功能介绍

提供可搜索帮助首页、章节目录、模块教程、完整流程、FAQ 和安全反馈指南；内容与 `docs/help` Markdown、前端 JSON 同步维护。

## 操作方法

1. 打开“系统设置 → 帮助与反馈”。
2. 输入关键词，如“简历上传”“实时语音”“搜索导入”。
3. 按分类或章节打开正文。
4. FAQ 中展开问题查看答案。
5. 反馈时选择类型，描述复现步骤、预期和实际结果。

## 使用场景

首次上手、功能排障、安全边界确认和问题反馈。

## AI 工作流

帮助搜索是本地关键词过滤，不把问题发给外部 AI；反馈内容不会自动发送到第三方。

## 用户输入与输出

- 输入：搜索词、章节选择、反馈分类和问题描述。
- 输出：教程正文、FAQ 答案和可复制诊断摘要。

## 注意事项

反馈前删除个人隐私和密钥；截图可能包含简历、邮箱、公司或岗位信息；标记“规划中”的能力不能使用。

## 常见问题

- **帮助内容离线可用吗？** 文档和 JSON 随前端构建，可本地阅读。
- **反馈会自动发送吗？** 当前只生成本地摘要，不代表已提交给客服。
