# 模块 4：JD 岗位匹配

**当前状态：可用。**

## 功能介绍

比较职位 JD 和简历，生成 0–100 分、已匹配关键词、缺失关键词、理由和改进建议，并保存分析记录。

## 操作方法

1. 在“职位搜索”新增或导入真实岗位并检查完整 JD。
2. 在“简历管理”准备对应简历。
3. 打开“JD 匹配”，选择职位和简历。
4. 运行匹配并阅读分数、命中项、缺口和理由。
5. 将分析用于简历优化、沟通和面试准备。

## 使用场景

在投递前筛选职位；比较多份简历；找出需要补充证据的技能。

## AI 工作流

JD + 简历 → 透明关键词和规则分析 → 分数与证据 → 保存匹配记录。当前重点是可解释结果，不代表招聘成功概率。

## 用户输入与输出

- 输入：职位记录、JD 文本和简历正文。
- 输出：匹配分、关键词覆盖、差距、理由和建议。

## 注意事项

JD 为空会降低可信度；分数仅供排序；薪资、黑名单和平台安全规则由投递模块另行硬校验。

## 常见问题

- **分数高为何仍不能投？** 可能触发薪资、黑名单或经历硬规则。
- **如何提高准确度？** 补充完整 JD 和真实简历文本。
