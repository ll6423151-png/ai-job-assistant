# 快速开始

## 1. 注册与登录

1. 打开欢迎页并选择注册。
2. 输入 QQ 邮箱，获取六位验证码。
3. 在 5 分钟内输入验证码并设置至少 8 位、同时含字母和数字的密码。
4. 登录后进入工作台；验证码 60 秒内不能重复发送。

## 2. 完成用户中心

填写姓名、求职城市、目标岗位、最低月薪、学历和经验。新账号默认无个人资料。最低月薪即使填写低于 3000，自动投递仍执行 3000 元硬门槛。

## 3. 准备主简历

上传 PDF、DOCX、TXT 或 Markdown；检查解析文本，设置为主简历、状态为“可投递”。扫描版 PDF 需要先转成可复制文字。

## 4. 获取职位

可以手工新增职位，或在电脑 Edge/web-access 已连接时搜索智联并导入。手机端通过后端复用电脑 Edge；每次只处理一页、最多五个候选。

已导入职位固定每页显示 7 条，可用页码和上一页/下一页切换。需要整理列表时，可以先勾选若干职位并清除，也可以二次确认后清除当前账号的全部智联导入职位；手工录入职位不会被“全部清除导入岗位”影响。清除会把岗位归档并从列表隐藏，历史匹配、沟通和投递快照继续保留；误操作后可点击“撤销上次清除”。

同一账号在相同搜索条件下再次遇到已判定不符合的同一岗位时，软件会在本地跳过后续 JD 读取和分析，并显示“历史跳过”数量。它不是公司黑名单，不影响同公司的其他岗位；需要重新评估时可清除当前条件记录。

## 5. 匹配、优化和沟通

选择职位和简历生成匹配报告；根据证据创建优化草稿，人工核对后应用；生成沟通内容并编辑、审核定稿。

## 6. 面试与投递

进行模拟面试并复盘报告。投递前逐条查看岗位、简历和沟通快照，勾选确认后提交。验证码、扫码和最终确认由用户完成。

## 7. 跟进结果

在投递记录中维护筛选中、面试、Offer、未通过等状态，设置跟进时间；在数据统计查看漏斗和趋势。
