欢迎使用 CareerPilot AI。帮助中心依据当前代码、接口和已验证能力编写；标记为“规划中”的功能当前不可使用。

## 建议阅读顺序

首次使用请依次阅读“软件简介 → 快速开始 → 简历管理 → JD 匹配 → 投递记录 → 完整流程”。涉及智联招聘时，先阅读系统设置和投递安全边界。

## 当前重要边界

- 当前只接入智联招聘；手机搜索依赖本机服务和电脑 Edge/web-access 桥接。
- 招聘平台登录、验证码和扫码由用户手动完成。
- 外部投递必须逐条预览和确认；系统不执行无人确认的批量投递。
- 公司“敏思跃动”在黑名单中；月薪低于 3000 元、无薪或无法确认达到 3000 元的岗位不会进入自动投递候选。
- 本地确定性 AI 可用；OpenAI Provider 未配置密钥时不会发起真实 OpenAI 请求。

## 反馈

在软件“系统设置 → 帮助与反馈”中选择问题类型并复制诊断摘要。请勿在反馈中粘贴密码、验证码、Cookie、QQ 邮箱 SMTP 授权码或招聘平台账号信息。
