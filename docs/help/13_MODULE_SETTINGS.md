# 模块 11：系统设置

**当前状态：本次新增基础设置入口；高级管理功能未实现。**

## 功能介绍

集中查看账号、安全、平台连接、环境状态和帮助与反馈。个人求职资料仍在“用户中心”维护。

## 操作方法

1. 打开“系统设置”。
2. 查看当前账号与数据隔离说明。
3. 在“帮助与反馈”搜索教程和 FAQ。
4. 智联搜索前确认电脑 Edge/web-access 已连接。
5. 使用顶部退出按钮结束登录。

## 使用场景

排查无法登录、邮件、手机连接、Edge 桥接、权限和版本问题。

## AI 工作流

设置本身不调用 AI。AI Provider、SMTP、数据库和域名由部署环境配置；普通用户不应输入系统 SMTP 授权码。

## 用户输入与输出

- 输入：帮助搜索词、反馈类型和问题描述。
- 输出：对应章节、FAQ、诊断建议和安全提醒。

## 注意事项

不要在反馈中粘贴密码、验证码、Cookie、API Key 或 SMTP 授权码；管理员后台、审计日志和细粒度权限当前暂停。

## 常见问题

- **如何改求职资料？** 去“用户中心”。
- **如何退出？** 点击顶部账号区域的退出图标。
- **为何没有管理员设置？** 不属于当前 MVP。
