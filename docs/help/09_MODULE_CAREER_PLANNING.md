# 模块 7：AI 职业规划

**当前状态：规划中，尚无独立页面、模型或 API。**

## 计划功能介绍

未来可基于用户资料、简历、目标岗位、匹配和投递结果生成阶段性求职计划、技能差距和行动建议。

## 当前替代操作

1. 在用户中心填写目标岗位、城市、薪资、学历和经验。
2. 用职位搜索收集代表性岗位。
3. 用 JD 匹配观察高频缺口。
4. 用数据统计查看投递转化。
5. 将结论手工记录在简历备注或投递备注中。

## 使用场景

选择求职方向、比较岗位族、规划学习优先级和调整投递策略。

## 计划 AI 工作流

用户资料 + 代表性 JD + 匹配历史 + 投递漏斗 → 高频技能与瓶颈 → 用户确认的阶段计划。当前系统不会自动执行此流程。

## 用户输入与输出

- 计划输入：职业目标、时间范围、技能自评和历史数据。
- 当前输出：无独立规划报告；只能组合现有模块自行判断。

## 注意事项

不要把规划建议当作录用承诺；涉及培训付费、转行或薪资决策时应结合真实市场和人工咨询。

## 常见问题

- **页面在哪里？** 当前没有独立页面。
- **能否自动生成学习路线？** 尚未实现，避免虚构已上线能力。
