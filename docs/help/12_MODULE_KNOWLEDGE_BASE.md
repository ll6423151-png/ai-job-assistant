# 模块 10：AI 知识库

**当前状态：规划中，尚无独立知识库、向量数据库或检索 API。**

## 计划功能介绍

未来可整理用户允许保存的面试复盘、岗位关键词、求职方法和常用沟通模板，并提供有来源的检索。

## 当前替代操作

1. 使用简历备注保存版本说明。
2. 使用匹配记录保存岗位差距。
3. 使用沟通内容保存审核模板。
4. 使用面试历史保存问答和报告。
5. 使用帮助中心查阅固定产品知识。

## 使用场景

复用高质量回答、沉淀行业关键词、准备常见面试题。

## 计划 AI 工作流

用户授权资料 → 脱敏与切分 → 检索 → 有来源回答。当前没有此数据管线，不会声称已检索个人知识库。

## 用户输入与输出

- 计划输入：用户选择的资料和文档。
- 当前输出：只能查看现有记录和固定帮助文档。

## 注意事项

未来实现必须按用户隔离、支持删除和来源追踪；不得默认采集隐私或招聘平台凭据。

## 常见问题

- **能上传资料问答吗？** 当前不支持知识库问答。
- **面试历史算知识库吗？** 它是结构化历史记录，不是通用检索知识库。
