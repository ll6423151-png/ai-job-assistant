# 模块 1：首页 Dashboard

**当前状态：部分实现。** 当前没有独立 Dashboard 标签，工作台标题、项目状态侧栏和“数据统计”共同承担概览功能。

## 功能介绍

查看当前账号、模块入口、安全提示、岗位/投递/面试统计和最近活动。

## 操作方法

1. 登录后通过顶部标签切换模块。
2. 查看右侧项目状态和安全边界。
3. 进入“数据统计”查看漏斗、趋势和最近投递。
4. 手机端横向滑动标签栏查看更多入口。

## 使用场景

每天开始求职前检查待跟进记录；复盘一周投递数量、面试率和 Offer 状态。

## AI 工作流

本模块只汇总本地记录，不调用生成式 AI。数据来自职位、匹配、投递和面试记录。

## 用户输入与输出

- 输入：其他模块产生的业务记录。
- 输出：状态概览、统计指标、最近活动和提醒。

## 注意事项

统计结果取决于是否及时更新投递状态；它不是招聘平台的实时数据。

## 常见问题

- **为什么没有数据？** 先创建简历、职位和投递记录。
- **为什么数据与智联不同？** CareerPilot 只统计已导入和已记录的数据。
