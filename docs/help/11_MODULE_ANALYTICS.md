# 模块 9：数据统计分析

**当前状态：可用。**

## 功能介绍

只读汇总岗位、简历、匹配和投递记录，展示回复率、面试率、Offer 率、渠道/状态分布、14 天趋势和最近活动。

## 操作方法

1. 持续维护投递状态和跟进时间。
2. 打开“数据统计”。
3. 查看关键指标、状态分布、渠道分布和趋势。
4. 对照岗位匹配分和沟通策略调整后续投递。

## 使用场景

周复盘、渠道比较、跟进提醒、识别简历或面试瓶颈。

## AI 工作流

当前为确定性聚合，不使用生成式 AI：本地记录 → 统计计算 → 图表/表格。未来解释建议不得替代原始数据。

## 用户输入与输出

- 输入：投递状态、渠道、跟进、匹配和面试记录。
- 输出：漏斗比例、趋势、分布和最近活动。

## 注意事项

未更新状态会造成统计偏差；小样本比例不稳定；统计不会从智联自动补齐所有历史数据。

## 常见问题

- **为什么 Offer 率为 0？** 没有记录 Offer 状态或分母不足。
- **为什么渠道不准确？** 检查每条投递的渠道字段。
