# 模块 5：AI 求职沟通

**当前状态：可用；智联 PC 无消息框时需在 App 手动发送。**

## 功能介绍

结合职位、简历和匹配证据生成简洁、专业或温暖语气的沟通草稿，支持编辑、审核定稿和复制。

## 操作方法

1. 选择职位和简历。
2. 选择语气并生成草稿。
3. 核对公司、岗位、技能和联系方式。
4. 编辑后点击审核/定稿。
5. 在投递任务中关联定稿内容；平台无输入框时复制到智联 App。

## 使用场景

首次沟通、投递后补充说明、面试感谢和跟进提醒。

## AI 工作流

职位 + 简历 + 最近匹配证据 + 语气 → 草稿 → 用户编辑 → 审核定稿 → 投递后尝试发送或手工复制。

## 用户输入与输出

- 输入：职位、简历、语气和人工修改。
- 输出：草稿、定稿状态和任务中的沟通快照。

## 注意事项

未经审核不应发送；不要承诺不存在的经验；当前智联 PC 常提示下载 App，系统会标记 `manual_required`，不会伪造发送成功。

## 常见问题

- **为什么投递成功但沟通失败？** PC 页面没有真实消息输入框，请复制到 App。
- **为什么定稿后还能看到旧内容？** 已创建任务保存的是创建时快照。
