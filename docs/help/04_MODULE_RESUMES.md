# 模块 2：简历管理

**当前状态：可用。**

## 功能介绍

创建和管理简历版本，上传文件并解析文本，设置主简历与可投递状态，为匹配、优化、面试和投递提供统一来源。

## 操作方法

1. 打开“简历管理”。
2. 选择上传 PDF、DOCX、TXT 或 Markdown，或手工新增。
3. 检查标题、目标岗位、版本、正文和备注。
4. 将准备好的简历状态改为 `ready/可投递`。
5. 选择一份设为主简历；投递前再次核对平台上的实际简历。

## 使用场景

针对不同岗位保存不同版本；上传旧简历继续编辑；为模拟面试选择特定简历。

## AI 工作流

文件先本地解析为文本；优化模块读取简历快照和 JD 匹配差距生成草稿。AI 不应虚构经历。

## 用户输入与输出

- 输入：简历文件、标题、目标岗位、正文、备注和状态。
- 输出：结构化简历记录、解析文本、主简历标记和版本列表。

## 注意事项

扫描 PDF 可能没有可提取文字；不要上传身份证、银行卡或无关敏感材料；多用户数据相互隔离。

## 常见问题

- **上传后为空？** 将扫描版 PDF 转为可复制文本。
- **为什么不能自动投递？** 必须有一份主简历且状态为可投递。
- **平台简历与本地不同怎么办？** 最终以智联弹窗/默认简历为准并人工核对。
